# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2025-01-01

### Added
- Initial release
- **Agent** resource: Create, Get, Get Many, Update, Delete, Publish
- **Call** resource: Create Phone Call, Create Web Call, End Call, Get, Get Many
- **Phone Number** resource: Create, Get, Get Many, Update, Delete
- **Batch Call** resource: Create, Get, Get Many, Start, Pause, Resume, Cancel
- **Analytics** resource: Get Summary, Get Calls by Day
- **Trigger**: Watch for completed calls, Watch for batch call status changes
- Bearer token authentication with configurable base URL
- `usableAsTool: true` for AI agent integration
